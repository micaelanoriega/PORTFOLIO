AUTENTICA

En este proyecto se ha desarrollado el diseño de un sitio web utilizando Figma, inicialmente optimizado para formato desktop, con la intención de adaptarlo a otros dispositivos. El objetivo fue crear un prototipo funcional que integre las necesidades de los usuarios, respetando principios heurísticos y priorizando una experiencia de usuario fluida. Para lograr esto, se realizó una investigación de usuario y secundaria, en la que se analizó a la competencia y a los user persona. El diseño sigue la metodología de DCU y está orientado a una empresa especializada en calcetines que requiere una plataforma de comercio electrónico para exhibir y vender sus productos. El sitio web incluye toda la experiencia de compra, desde la navegación hasta la finalización de la transacción, garantizando un diseño adaptable.
